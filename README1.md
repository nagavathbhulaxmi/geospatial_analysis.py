# Geospatial Data Analysis

## Overview

This project analyzes geographical sales data to identify high-demand regions with low business presence.

## Technologies

- Python
- Pandas
- Folium
- GeoPandas
- Matplotlib

## Features

- Interactive Maps
- Sales Analysis
- Business Expansion
- Customer Distribution
- Demand Visualization

## Output

- Business Map
- Expansion Recommendation
- High Demand Regions
